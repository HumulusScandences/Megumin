前言
-----------

   由于腾讯并不开放机器人API接口，任何机器人都有被封号的风险。为尽量减少服务中断的风险，在群中使用机器人需先申请。

   **请注意, 此申请仅适用于Dice!官方机器人(Kokona系列)**

申请群中使用权限
-----------------------

    您必须先阅读 **用户协议** 再进行申请。请先 **将骰娘拉入群中** 再 **提交申请** ，申请方法有两种 (**选择其一**):

    1.加入群941980833或624807593(已满)，在群中发布消息，格式为"入群申请:群号:[群号] 原因:[原因] 骰娘:[1号/2号/3号] 窝已仔细阅读并保证遵守Dice!云掷骰服务用户协议"（推荐）

    2.私聊机器人，输入"!info 入群申请:群号:[群号] 原因:[原因] 窝已仔细阅读并保证遵守Dice!云掷骰服务用户协议"，注意,!info和入群申请之间必须有一个空格(**不加双引号**)。

    未拉骰娘的申请不会被通过，同时请尽量不要重复申请。窝们不对申请通过的时间做任何保证。申请成功后，申请成功的提示会被发送到群中。申请原因应注明游戏类型(COC/DND etc.)以及模组名称(如果已经确定)。

    很抱歉由于管理都是学生党，申请处理速度可能很慢，窝们对可能造成的不便表示抱歉。如果您急切需要使用骰子，建议参考搭建说明自行搭建，异或尝试其他非官方机器人。
